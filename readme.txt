Generates redirect code for .htaccess file

== Description ==

Using this plugin, you can create redirects for any post types or taxonomies.

How to use:

* In the "Tools" menu, find the "Htaccess Redirect" menu item.
* Select the post types and taxonomies for which you want to create redirects.
* In the "New site domain" field, enter the URL of the site to which the redirect will be made.
* Click the "Generate" button.
* Insert the generated redirects at the beginning of the .htaccess file.


== Installation ==

1. Unzip and upload the "plance-redirects-for-htaccess" to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the "Htaccess Redirect" page in the "Tools" menu.
